# Time-Series Forecasting

This repository provides state-of-the-art neural network models for time-series forecasting. It includes robust implementations of LSTM, GRU, and Transformer-based architectures designed for accurate sequential data prediction.

For further details and related research, check the publications here: [DOI: 10.3390/nu15051199](https://doi.org/10.3390/nu15051199)

The implementations in this repository were inspired by and reference the following resources:

- [Time Series Prediction with LSTM Recurrent Neural Networks in Python/Keras](https://machinelearningmastery.com/time-series-prediction-lstm-recurrent-neural-networks-python-keras/)
- [Keras Example: Time Series Classification with Transformers](https://keras.io/examples/timeseries/timeseries_classification_transformer/)
